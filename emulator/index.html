Pocket Deck Emulator

Try Pocket Deck apps in the browser. Keyboard, D-pad, and slider on the device image are clickable. Drag the D-pad center ring to rotate the dial (used by the kitchen timer).

For Mac user, use Google Chrome. Emulator does not support Safari. Set fast key repeat to get better response.

Screen switch buttons are not supported on emulator. In real device, it provides 10 virtual screens to switch between applications.
