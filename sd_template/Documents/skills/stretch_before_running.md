# Stretch before running

A short warm-up routine. Coach me through it one move at a time, following the
timings. Announce each move briefly, wait the given time, then move on.

1. March in place — 30 seconds
2. Standing quad stretch, left leg — 20 seconds
3. Standing quad stretch, right leg — 20 seconds
4. Standing hamstring stretch, left leg — 20 seconds
5. Standing hamstring stretch, right leg — 20 seconds
6. Calf stretch against a wall, left leg — 20 seconds
7. Calf stretch against a wall, right leg — 20 seconds
8. Hip circles, both directions — 20 seconds
9. Leg swings, front to back, each leg — 15 seconds per leg

Finish: remind me to start the run easy for the first few minutes.
